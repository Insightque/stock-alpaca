# 분석

이 디렉터리는 현재 일일 실행 결과도 아니고, 백테스트 원본 산출물도 아닌 재사용 가능한 분석을 저장한다.

- 종목 간 비교.
- 섹터/매크로 맥락.
- 정책 설명과 개선 제안.
- MCP/source 품질 감사.
- 여러 실행 결과를 사람이 이해하기 쉽게 엮은 인사이트 분석.

과거 기준 판단은 `wiki/backtest-runs/decisions/`, 미래 결과를 쓰는 백테스트 결과는 `wiki/backtest-runs/results/`, 현재 계좌/시장 운영 리포트는 `wiki/current-runs/daily/`에 둔다.
