# 현재 실행 기록

이 디렉터리는 현재 계좌와 현재 시장을 대상으로 실행한 workflow 결과만 저장한다.

- `daily/`: 현재 시점의 일일 trading workflow 결과.

저장 기준은 문서가 사람이 읽기 쉬운지 여부가 아니라 목적과 정보 사용 방식이다.

- 현재 계좌/현재 시장의 일일 실행 결과: `wiki/current-runs/daily/`.
- 과거 기준 의사결정: `wiki/backtest-runs/decisions/`.
- 과거 의사결정의 결과 평가나 정책 백테스트: `wiki/backtest-runs/results/`.
- 종목 간 비교, 정책 설명, MCP/source 감사, 재사용 가능한 해설: `wiki/research-notes/analyses/`.
- 실제 체결된 paper 거래의 사후 회고: `wiki/trade-ledger/reviews/`.
