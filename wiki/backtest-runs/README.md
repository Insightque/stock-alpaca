# 백테스트

이 디렉터리는 과거 기준 모의 의사결정, 그 의사결정의 사후 회고, 여러 의사결정을 묶은 정책 검증 결과를 한곳에 모아 관리한다.

핵심 구분은 `시뮬레이션 vs 백테스트`가 아니라 `미래 정보 금지 판단 기록 vs 미래 결과를 쓰는 평가 기록`이다.

## 하위 구조

- `decisions/`: 과거 특정 시점 기준 모의 의사결정. 추천 문서에는 기준 시점 이후 가격, 뉴스, 결과를 넣지 않는다.
- `results/`: 미래 결과를 쓰는 모든 백테스트 결과. 단일 의사결정의 1D/5D/20D 회고와 여러 날짜를 묶은 정책 검증을 모두 여기에 둔다.
- `datasets/`: backtest 산출 데이터의 위치 안내. 대용량 원천/계산 JSON은 원칙적으로 `wiki/evidence-store/sources/`에 둔다.

기계가 읽는 실행 provenance는 `wiki/evidence-store/run-manifests/`에 둔다.
