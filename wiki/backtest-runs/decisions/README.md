# 백테스트 의사결정 기록

이 디렉터리는 특정 과거 시점을 기준으로 당시 알 수 있었던 정보만 사용해 만든 모의 의사결정 기록을 저장한다.

- 추천 문서는 미래 가격 정보를 포함하지 않는다.
- 실제 주문은 제출하지 않는다.
- 파일명은 `YYYY-MM-DD-historical-decision.md` 형식을 사용한다.
- 성과 평가는 `wiki/backtest-runs/results/`에 별도 문서로 저장한다.
- 기준 시점 이후 가격, 뉴스, 결과가 들어가는 모든 문서는 `wiki/backtest-runs/results/`에 저장한다.
