# MCP 보강 과거 추천 검증 - 2026-05-11~2026-05-15

## 목적

- 기존 v2 규칙에 Alpha Vantage 실적 데이터와 MCP 데이터 공백 정보를 반영해 같은 out-of-sample 검증셋을 다시 평가한다.
- 추천 단계는 기준일 이후 가격을 사용하지 않는다는 전제로 기록한다.
- 실제 주문은 제출하지 않았다.

## 보강 규칙

- 실적 서프라이즈는 긍정 촉매로 인정하되, 발표 후 3~5거래일 누적 상승률이 과도하면 신규 매수 점수를 낮춘다.
- 보강 MCP에서 뉴스가 0건이거나 macro/filing 데이터가 비어 있으면 긍정 신호를 추가하지 않는다.
- 급등 직후의 IONQ/AMD 같은 후보는 1D 급등과 거래량 급증이 동시에 있으면 다음 조정까지 대기한다.
- 대형 AI 핵심 후보는 방어적으로 1자리만 유지하고, 나머지는 방어/품질/조정 후 재가속 후보로 분산한다.
