# 백테스트 결과

이 디렉터리는 기준 시점 이후의 실제 결과를 사용하는 모든 백테스트 결과를 저장한다.

- 단일 과거 의사결정의 1D/5D/20D 사후 평가.
- 1년/6개월/분기 단위 policy backtest.
- 단타/장타 전략 validation.
- event study와 정책 개선 후보 검증.
- hit rate, 초과수익, drawdown, 비용 반영 여부, 데이터 공백을 포함하는 성과 요약.

개별 과거 기준 판단 자체는 `wiki/backtest-runs/decisions/`에 둔다. 판단 문서에는 미래 결과를 넣지 않고, 결과 평가는 이 디렉터리에만 둔다.
