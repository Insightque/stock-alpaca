# 거래 장부

이 디렉터리는 실제 paper 주문, 포지션, 거래 회고의 생애주기 기록을 저장한다.

- `orders/`: 실제 paper 주문과 dry-run 주문 계획.
- `positions/`: 실제 paper 계좌/포지션 스냅샷.
- `reviews/`: 종목별 또는 포트폴리오 단위 거래 회고.
- 신규 체결 직후에는 판단을 서두르지 않고 `회고 대기`로 표시할 수 있다.
- 청산된 거래는 최종 회고를 남긴다.
- 보유 중인 거래는 필요할 때 중간 회고를 남긴다.

회고는 수익 여부만 보지 않는다. 당시 정보로 좋은 판단이었는지, 어떤 리스크를 잘 보았는지, 무엇을 놓쳤는지, 추천 정책에 어떤 교훈을 남길지를 기록한다.

과거 추천 회고와 정책 백테스트 결과는 `wiki/backtest-runs/results/`에 둔다. 원 추천 문서를 수정하지 않고 별도 문서에서 미래 가격과 벤치마크 대비 결과를 평가한다.
