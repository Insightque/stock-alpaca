# 거래 회고

이 디렉터리는 체결된 paper 거래의 사후 검토를 저장한다.

- `trades/`: 종목별 또는 포트폴리오 단위 거래 회고.
- `decisions/`: 과거 시점 추천 시뮬레이션의 1D/5D/20D 성과 회고.
- 신규 체결 직후에는 판단을 서두르지 않고 `회고 대기`로 표시할 수 있다.
- 청산된 거래는 최종 회고를 남긴다.
- 보유 중인 거래는 필요할 때 중간 회고를 남긴다.

회고는 수익 여부만 보지 않는다. 당시 정보로 좋은 판단이었는지, 어떤 리스크를 잘 보았는지, 무엇을 놓쳤는지, 추천 정책에 어떤 교훈을 남길지를 기록한다.

과거 추천 회고는 원 추천 문서를 수정하지 않고 별도 문서에서 미래 가격과 벤치마크 대비 결과를 평가한다.
